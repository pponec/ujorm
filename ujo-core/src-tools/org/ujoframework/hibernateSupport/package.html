Benefit

The UJO object can improve the performance of the Hibernate query language mainly the SELECT statement. In my tests, I measured a 1.75 times increase of performance. The test was performed on the Hibernate 3.3.1.GA with an EhCacheProvider and SpeedUjo object implementation.

Samples

There is possible to change a default Hibernate Accessor in HBM file by the next sample: 
<hibernate-mapping
  package="performance.test"
  default-access="org.ujoframework.hibernateSupport.UjoPropertyAccessor"
  >
  <class name="MyClass" table="MY_CLASS" >
