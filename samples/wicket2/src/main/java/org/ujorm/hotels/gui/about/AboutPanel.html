A demo application based on the Ujorm, Wicket and Spring frameworks is running on the Java runtime .

Description

The application is built on the class model: the entity Booking is a relation of type M:N between the Hotel and the Customer. The customer can therefore book an accommodation in more hotels, and one hotel reservation holds more customers. To simplify, we do not care about the capacity of the hotel. To make the data model more interesting, we should add next entity called City, which is an attribute of the Hotel, however the project have no administration for the City entity. All system and customer parameters are saved in the ParamValue entity.
For more information see the full original article.
