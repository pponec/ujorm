Demo hotels

A demo application based on the Ujorm, Wicket and Spring frameworks.

Running the project

The project have got the Maven structure, so you can call from command line:
  • download project from Source Forge
  • unzip sources to a new directory and go there
  • run server: mvn jetty:run
  • and open application: firefox localhost:8080
The Java 6.0+ is required for complilaction the souces.

How to open this project?

Use a one of the preferred tools to open the project:
  • NetBeans IDE,
  • InteliJ IDEA,
  • Eclipse with some Maven 2 pluggin,
  • or use the Maven 2 toolkit if you like a console command line
