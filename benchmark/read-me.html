How to open this project?

The Java 6.0 is required to compile of the project souces, the Java 5+ is required to runtime.
Use a one of the preferred tools to open the project:

  • NetBeans IDE,
  • InteliJ IDEA,
  • Eclipse with some Maven 2 pluggin, or use the
  • Maven 2 toolkit if you like a console command line

How to create a 'benchmark' database for the PostgreSql on Ubuntu?

Type from a command line:

  • sudo apt-get install postgresql pgadmin3 -y
  • sudo -u postgres createdb benchmark
  • sudo -u postgres psql -s benchmark
  • > CREATE USER sa PASSWORD 'sa';
  • > GRANT ALL PRIVILEGES ON DATABASE benchmark TO sa;
  • > Ctrl-D
