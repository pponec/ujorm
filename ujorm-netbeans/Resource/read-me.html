Ujorm

version ${VERSION}

Ujorm provides special objects with different architecture from JavaBeans.
The objects have got an easy object introspection or simple persistence
to several formats include XML. Since 0.90 is included an ORM implementation called Ujorm.

Top Features

  • map based architecture with a value type safe solution builded on two interfaces
  • an easy object introspection without a heavy PropertyDescriptor implementation
  • unique object-relation mapping (ORM) implementation is available
  • efficient high speed XML serialization or
  • persistence to CSV or Resource bundle format
  • JTable component support
  • JavaBean support
  • easy to use lightweight ORM module called Ujorm
  • features are checked by JUnit tests
  • very light weight framework with no dependencies
  • open source

How to open this project?

The Java 6.0+ is required to compile of the project souces, the Java 5++ is required to runtime.
Use a one of the preferred tools to open the project:

  • NetBeans IDE,
  • InteliJ IDEA,
  • Eclipse with some Maven 2 pluggin, or use the
  • Maven 2 toolkit if you like a console command line
